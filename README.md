# PyArtist — Minimalist Watercolor Artist Portal

A dual-module platform for a watercolor artist:

- **Website** (`/website`): minimalist static site that consumes `gallery.json` and Markdown files.
- **Backoffice** (`/backoffice`): Flask web app for artwork management, image optimization, and FTP deployment.

## Stack

| Module | Technology |
|---|---|
| Website | HTML5, Tailwind CSS, Alpine.js |
| Backoffice | Python 3.10+, Flask, Flask-SQLAlchemy |
| Database | Turso (libSQL/SQLite) |
| Auth | Google OAuth2 (Authlib) |
| Image Engine | Pillow |
| Deploy | Render.com (backoffice) + FTP (website) |

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cp .env.example .env
# Fill in .env with your credentials

cd backoffice
flask run
```

## Workflow

1. The artist logs into the backoffice via Google Login.
2. Uploads an image with title, category, and description.
3. The system optimizes the image (max 1920px) and generates a thumbnail (400px).
4. `website/data/gallery.json` is regenerated.
5. The website is updated via FTP with the new images and JSON.
